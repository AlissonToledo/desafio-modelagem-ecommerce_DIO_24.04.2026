Projeto de Banco de Dados – E-commerce

Este projeto apresenta a modelagem conceitual de um banco de dados para um sistema de E-commerce. O diagrama representa as principais entidades e relacionamentos necessários para o gerenciamento de clientes (PF ou PJ), produtos, categorias, fornecedores, estoque, pedidos, itens do pedido, pagamentos e entregas.
O objetivo é estruturar as informações de forma organizada utilizando conceitos de modelagem de dados e bancos de dados relacionais, garantindo a integridade e o relacionamento adequado entre as entidades do sistema.
